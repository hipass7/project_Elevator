필요한 CAN 통신 목록

1) panel <-> main
    send)
        - button (bool)
    receive)
        - elevator 위치 (vector)

2) elevator <-> main
    send)
        - button (int)
    receive)
        - 목적지 (vector)
        
CAN 사용법
1) #if 0을 1로 변경

2) 아래 명령어 수행
sudo modprobe vcan
sudo ip link add dev vcan0 type vcan
sudo ip link set up vcan0

3) rx_id 1027로 0x01을 보내는 예시
cansend vcan0 403#01


윈도우에서 빌드할 때
cmake ..
cmake --build ./
